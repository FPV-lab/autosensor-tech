Build by Size — Quick Reference

Tiny Whoop Build (65mm)

AIO board (FC+ESC+VTX+RX), 0802 motors, 31mm props, 1S battery. Total: $80-120.

3-Inch Micro Build

20x20 stack, 1404 motors, 3" props, 4S 450-650mAh, analog or HDZero. Total: $180-280.

5-Inch Standard Build

30.5mm stack (F405+55A ESC), 2207 1900KV motors, 5" props, 6S 1300mAh, DJI O4, ELRS. Total: $470-550.

7-Inch Long Range Build

30.5mm stack, 2806 1500KV motors, 7" props, 6S 3000mAh or Li-Ion, 900MHz ELRS, GPS. Total: $500-700.
