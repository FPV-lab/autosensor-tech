💻 Betaflight & Software

Flight controller firmware, configuration tools, and tuning guides

What is Betaflight?

Betaflight is the most popular open-source flight controller firmware for FPV drones. It runs on STM32 processors and provides real-time stabilization, PID control, OSD overlay, GPS rescue, and much more. As of 2026, Betaflight 4.5 is the latest stable release.

Key Fact: Betaflight supports 95%+ of all FPV flight controllers on the market. If you're flying FPV, you're almost certainly running Betaflight.

Betaflight Configurator

The Betaflight Configurator is a Chrome/standalone app used to configure your flight controller. It connects via USB or Bluetooth.

Initial Setup (Fresh Build)

Step 1: Flash Firmware

  1. Connect FC via USB (boot button if needed)
  2. Open Configurator → "Firmware Flasher" tab
  3. Select your FC target (check FC label)
  4. Choose latest stable release (4.5.x)
  5. Click "Load Firmware" → "Flash"

Step 2: Ports Configuration

PeripheralUARTFunction
ELRS ReceiverUART 1 (or 2)Serial RX ✅
DJI O3/O4 Air UnitUART 2 (or 3)VTX (MSP + DisplayPort)
GPS ModuleUART 3GPS (57600 baud)
HDZero/WalksnailUART 2VTX (MSP + DisplayPort)

Step 3: Configuration Tab

  • Gyro/PID Loop: 8kHz/8kHz (default)
  • Mixer: Select your frame type (Quad X)
  • Receiver: Serial-based, CRSF protocol (for ELRS)
  • Arming: Set AUX1 switch for arm/disarm
  • Failsafe: Set to "GPS Rescue" (if GPS installed) or "Drop"

Step 4: Receiver Setup

  • Bind ELRS receiver to your transmitter
  • Go to "Receiver" tab → verify channel mapping
  • Roll/Pitch/Yaw/Throttle should respond to sticks
  • Set channel map: AETR1234 (standard)

Step 5: Motors Test

⚠️ Remove propellers before testing motors!
Go to "Motors" tab → enable motor test sliders → verify each motor spins correctly and in the right direction.

PID Tuning

PIDs (Proportional-Integral-Derivative) control how aggressively the drone responds to your stick inputs. Getting PIDs right makes the difference between a shaky, unstable drone and a locked-in, smooth flyer.

Default PIDs (Betaflight 4.5)

AxisPID
Roll428530
Pitch469032
Yaw45900

What Each PID Does

  • P (Proportional): How aggressively it corrects errors. Too high = oscillation, too low = sluggish.
  • I (Integral): Corrects long-term drift. Too high = slow wobble, too low = drifts in wind.
  • D (Derivative): Dampens corrections. Too high = hot motors, too low = overshoot.

Quick Tuning Tips

  • Oscillation? → Lower P by 5-10 points
  • Sluggish response? → Raise P by 5-10 points
  • Drifts in wind? → Raise I by 10-20 points
  • Hot motors? → Lower D by 5 points
  • Overshoot on flips? → Raise D by 3-5 points

Filters

Filters remove noise from gyro data before it reaches the PID controller. Proper filtering keeps motors cool and flights smooth.

OSD (On-Screen Display)

OSD overlays flight data on your FPV feed. Essential for monitoring battery voltage, flight time, and GPS coordinates.

Essential OSD Elements

  • Battery Voltage: Always visible
  • Flight Timer: Track remaining battery
  • RSSI/Link Quality: ELRS shows LQ
  • Throttle Position: Helps manage battery
  • GPS Coordinates: Essential for long range
  • Artificial Horizon: Useful for orientation

GPS Rescue Mode

GPS Rescue is Betaflight's built-in failsafe. When activated, the drone automatically flies back to your home position and lands.

Setup Requirements

  • GPS module (BN-220, BN-880, or Matek M10Q)
  • Configured UART for GPS
  • Minimum 8 satellites before arming
  • Set failsafe to "GPS Rescue"

ExpressLRS Firmware

ELRS receiver and transmitter firmware needs regular updates. Use the ELRS Configurator (web or desktop) to flash new firmware.

ELRS Web Flasher

  • URL: flasher.expresslrs.org
  • Select TX/RX target
  • Choose binding phrase
  • Set packet rate (250Hz recommended for range)
  • Flash via WiFi or USB
